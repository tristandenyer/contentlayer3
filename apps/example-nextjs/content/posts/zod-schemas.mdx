# Schema Validation with Zod

Zod is a TypeScript-first schema validation library. It provides an expressive API for defining data schemas with automatic type inference.

With Zod, you can validate data from APIs, forms, and user input. The rich error messages help you understand validation failures.

CL3 uses Zod as its schema system, making it the perfect choice for content validation and type safety across your application.
