# Next.js Guide for Modern Web Apps

Next.js is the React framework for production. It provides a great developer experience and enables deploying to the edge.

Key features include file-based routing, API routes, and automatic optimization. The App Router (introduced in Next.js 13) brings React Server Components to your application.

With CL3, you can manage your content at runtime and leverage Next.js features like ISR and on-demand revalidation.
